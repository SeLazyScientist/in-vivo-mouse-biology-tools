Backcrossing planner

When you import a strain on a different (or mixed) background, you usually want it congenic to the strain your phenotype is calibrated against. This planner shows the percent donor genome remaining at each generation, and how speed congenic shortens the timeline.

Generations to congenic

Standard backcross: each generation halves the residual donor genome (50% → 25% → 12.5% → ...). Speed congenic exploits the variance — the lucky pup who inherited fewer donor segments is selected each round, reaching N5 equivalents by N3-N4.

About this tool

This is a planning aid, not a substitute for reading the original strain papers and the JAX or MGI datasheets. Phenotypes drift, sublines diverge, and what worked in one institution's colony will not always reproduce in another.

Primary references for the calculation logic

  • Mendelian segregation: standard textbook (e.g. Nagy et al., Manipulating the Mouse Embryo).
  • Backcross genome residual: Silver, Mouse Genetics: Concepts and Applications (1995), updated by Wakeland et al., Immunol Today (1997) for speed congenic.
  • Cohort sizing under negative binomial: standard normal approximation, n ≈ N/p + 1.645 · √(N(1-p))/p for one-sided 95% upper bound.
